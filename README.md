# Adelante CRM

**Adelante CRM** — це преміальна система управління бізнесом, розроблена для салонів краси, студій та сервісних центрів. Система поєднує в собі потужний функціонал з витонченим та сучасним інтерфейсом.

## ✨ Основні можливості

- **📅 Розумний Календар**: Інтуїтивне планування записів, керування розкладом майстрів та швидке створення візитів.
- **🔍 Центр Аудиту (Огляд)**: Глибока аналітика історії записів, відгуків та змін даних. Повний контроль за діями персоналу.
- **🖼 Медіа-архів**: Збереження фото робіт безпосередньо в картку запису з підтримкою Drag & Drop.
- **👥 Клієнтська база**: Детальні профілі клієнтів, історія візитів та інтеграція з месенджерами.
- **💰 Фінанси та Склад**: Облік операцій, робота з касами та контроль залишків товарів.
- **⚙️ Гнучкі налаштування**: Керування ролями, правами доступу та параметрами закладу.

## 🛠 Технологічний стек

- **Frontend**: Next.js 14, React, TypeScript
- **Стилізація**: Vanilla CSS (Custom UI System)
- **Стан**: Zustand (Domain-driven stores)
- **Іконки**: Lucide
- **API**: Axios + Interceptors (Mock support enabled)
